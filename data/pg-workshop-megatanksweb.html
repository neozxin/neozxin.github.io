MegaTanksWeb

MegaTanksWeb is an ongoing game development project off my office hours, the game is from the classic Nintendo FC Battle City.

Today, I have rewriten this game using HTML5/CSS3/JavaScript, the game supports keyboards and touch devices(single player only). I will continue to add more features and make it more enjoyable. I hope it can bring you more fun too.
